The solid Green Dot indicates that the recipient has a notification set for either today or tomorrow.

The Yellow Dot indicates that the recipient has a notification set sometime after tomorrow.

The Blinking Red Dot indicates that the recipient has no set notifications.

Use the Red X to delete recipients and notifications. A prerequisite for recipient deletion is an absense of notifications.
